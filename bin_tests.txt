Tests for BinItems and Bins

  >>> from warehouse import BinItem, Bin, Warehouse


BinItems
========

BinItems consist of sku and quantity attributes, which are both displayed
when a BinItem is printed.

  >>> item = BinItem('12345-27', 2136)
  >>> item.sku
  '12345-27'
  >>> item.quantity
  2136
  >>> print(item)
  SKU 12345-27: 2136


Bins
====

Warehouse
===

  >>> a_ware = Warehouse('W')
  >>> a_ware.name
  'W'
  >>> a_ware.bis
  []
  
Bin objects are containers that hold BinItems.

  >>> a_bin = Bin('A')
  >>> a_bin.name
  'A'
  >>> a_bin.contents
  []
  >>> print(a_bin)
  Bin A:

  >>> a_ware.addbi(a_bin)
  >>> b_bin = Bin('B')
  >>> a_ware.addbi(b_bin)
  >>> print(a_ware)
  Warehouse W:
    Bin A:
    Bin B:
  >>> a_ware.rmbi(b_bin)
  >>> print(a_ware)
  Warehouse W:
    Bin A:


  >>> a_ware.add(a_bin, BinItem('12345-45', 500))
  >>> print(a_bin)
  Bin A:
    SKU 12345-45: 500

Items should be inserted in order of sku number

  >>> a_ware.add(a_bin, BinItem('12345-27', 4320))
  >>> print(a_bin)
  Bin A:
    SKU 12345-27: 4320
    SKU 12345-45: 500
  >>> a_ware.add(a_bin, BinItem('12345-32', 29))
  >>> print(a_bin)
  Bin A:
    SKU 12345-27: 4320
    SKU 12345-32: 29
    SKU 12345-45: 500
  >>> a_ware.add(a_bin, BinItem('12345-15', 333))
  >>> print(a_bin)
  Bin A:
    SKU 12345-15: 333
    SKU 12345-27: 4320
    SKU 12345-32: 29
    SKU 12345-45: 500
  >>> a_ware.add(a_bin, BinItem('12345-97', 2343))
  >>> print(a_bin)
  Bin A:
    SKU 12345-15: 333
    SKU 12345-27: 4320
    SKU 12345-32: 29
    SKU 12345-45: 500
    SKU 12345-97: 2343

If an item which is already in the bin is added, only the quantity should
change.

If an item has 0 quantity, it should be deleted

  >>> a_ware.add(a_bin, BinItem('12345-32', 100))
  >>> print(a_bin)

    SKU 12345-32: 129
    SKU 12345-45: 500
